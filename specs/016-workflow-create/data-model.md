# Data Model: Create Workflow

**Feature**: 016-workflow-create
**Date**: 2026-03-22

---

## Overview

No new DynamoDB tables or schema changes. Workflows are written to the existing `workflows` table. This feature adds the write path (create) alongside the existing read paths.

---

## Existing DynamoDB Table: `workflows`

| Field | Type | Required | Notes |
|-------|------|----------|-------|
| `id` | String (PK) | ✅ | UUID v4, generated by backend |
| `name` | String | ✅ | Human-readable label |
| `index` | String | ✅ | Underlying index symbol (e.g., `DAX.I`) |
| `cfd` | String | ✅ | Tradeable CFD symbol (e.g., `GER40.I`) |
| `enable` | Boolean | ✅ | Default: `true` |
| `dry_run` | Boolean | ✅ | Default: `true` (safe default) |
| `is_us` | Boolean | ✅ | Default: `false` |
| `end_date` | String (optional) | ❌ | ISO date `YYYY-MM-DD`; must be future date if provided |
| `conditions` | List\<Map\> | ✅ | At least one condition required |
| `trigger` | Map | ✅ | Single trigger object |
| `created_at` | String | ✅ | ISO 8601 timestamp with Z suffix, set by backend |
| `updated_at` | String | ✅ | Same as `created_at` on creation, set by backend |

### Condition Map Structure

| Field | Type | Required | Notes |
|-------|------|----------|-------|
| `indicator.name` | String | ✅ | One of: `ma50`, `combo`, `bbb`, `bbh`, `polarite`, `zone` |
| `indicator.ut` | String | ✅ | One of: `daily`, `15m`, `h1`, `h4`, `weekly`, `monthly` |
| `indicator.value` | Number (optional) | ❌ | Required for `polarite` and `zone` (lower bound) |
| `indicator.zone_value` | Number (optional) | ❌ | Required for `zone` (upper bound) |
| `close.direction` | String | ✅ | `above` or `below` |
| `close.ut` | String | ✅ | Unit time for the close candle |
| `close.spread` | Number | ✅ | Tolerance in points (> 0) |
| `element` | String (optional) | ❌ | `close`, `high`, or `low` |

### Trigger Map Structure

| Field | Type | Required | Notes |
|-------|------|----------|-------|
| `ut` | String | ✅ | Unit time for the trigger candle |
| `signal` | String | ✅ | Always `breakout` — hardcoded by backend |
| `location` | String | ✅ | `higher` or `lower` |
| `order_direction` | String | ✅ | `buy` or `sell` |
| `quantity` | Number | ✅ | Must be > 0 |

## New Frontend TypeScript Interfaces

### Location: `frontend/src/services/api.ts`

```ts
export interface WorkflowIndicatorInput {
  name: string;         // "ma50" | "combo" | "bbb" | "bbh" | "polarite" | "zone"
  ut: string;           // "daily" | "15m" | "h1" | "h4" | "weekly" | "monthly"
  value: number | null;
  zone_value: number | null;
}

export interface WorkflowCloseInput {
  direction: string;    // "above" | "below"
  ut: string;
  spread: number;
}

export interface WorkflowConditionInput {
  indicator: WorkflowIndicatorInput;
  close: WorkflowCloseInput;
  element: string | null;  // "close" | "high" | "low" | null
}

export interface WorkflowTriggerInput {
  ut: string;
  location: string;     // "higher" | "lower"
  order_direction: string;  // "buy" | "sell"
  quantity: number;
}

export interface WorkflowCreateRequest {
  name: string;
  index: string;
  cfd: string;
  enable: boolean;
  dry_run: boolean;
  is_us: boolean;
  end_date: string | null;
  conditions: WorkflowConditionInput[];
  trigger: WorkflowTriggerInput;
}
```

Response type: existing `WorkflowDetail` interface (already defined in api.ts).

## Validation Rules

| Field | Rule |
|-------|------|
| `name` | Non-empty after stripping whitespace |
| `index` | Non-empty |
| `cfd` | Non-empty |
| `end_date` | If provided: valid `YYYY-MM-DD` date, must be ≥ today |
| `conditions` | Minimum 1 condition |
| `indicator.value` | Required (non-null) when `name = "polarite"` or `name = "zone"` |
| `indicator.zone_value` | Required (non-null) when `name = "zone"` |
| `close.spread` | Must be > 0 |
| `trigger.quantity` | Must be > 0 |
